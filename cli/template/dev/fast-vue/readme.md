* upload 保持不变  
* build 保持不变
* dev 不在对css增加前缀 和js 进行es6转es5
* dev:low 增加css前缀 和js 进行转换 对低版本进行兼容
* build:local 打包路径为本地路径，即不增加域名
* upload:view 上传路径为测试环境 ，但是 文件相对路径不变
* lint 全部代码格式化 git push 前请执行
